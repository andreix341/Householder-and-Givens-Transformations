# Householder and Givens Transformations Visualization

A Python GUI application for visualizing **Householder reflections** and **Givens rotations**, built with **Tkinter** and **Matplotlib**.

---

## Overview
This project demonstrates two fundamental orthogonal transformation techniques used in numerical linear algebra:

* **Householder Reflections**: Reflect a vector across a hyperplane defined by a Householder vector $v$, with an option to compute $v^{\perp}$ via two methods.
* **Givens Rotations**: Rotate a vector in the plane spanned by two basis vectors using cosine/sine parameters.

---

## Features
* Interactive GUI for inputting vectors and transformation parameters.
* Support for 2D and 3D visualizations.
* Toggle between Householder and Givens transformations.
* Method selection for computing perpendicular vectors (Householder).
* Real-time 3D plotting with Matplotlib.

---

## Requirements
* `numpy`
* `matplotlib`
* `tkinter` (included with Python)

## Transformations

### Householder Reflection
Reflects vector $x$ across a hyperplane, with reflection direction controlled by index $k$. Supports two methods for computing $v^{\perp}$:

1.  **Method 1**: Uses cross product with coordinate basis vectors.
2.  **Method 2**: Projects $x$ onto $v$ to find the perpendicular component.

**Lines:**
* **Blue Line**: The original vector $x$ (input vector), e.g., $(2, 5)$ with $k=1$.
* **Orange Line**: The reflected vector $x'$ after applying the Householder transformation.
* **Green Line**: The subspace spanned by $v$ (the reflection direction, $v = x - \|x\| e_k$).
* **Dotted Red Line**: The subspace spanned by $v^{\perp}$ (the hyperplane normal to $v$—the “mirror” of reflection).

**Comments:**
* The Householder reflection flips $x$ across the hyperplane orthogonal to $v$.
* $v^{\perp}$ defines the “mirror” (a line in 2D). The angle between $x$ and $x'$ is twice the angle between $x$ and $v^{\perp}$.
* It’s used in QR decomposition to zero out the subdiagonal elements of a matrix.

### Givens Rotation
Rotates vector $x$ in the $(i, k)$ plane using computed $\cos$ and $\sin$ values. Useful for zeroing out specific vector components.

#### 2D Visualization
<img width="470" height="458" alt="image" src="https://github.com/user-attachments/assets/50212929-1650-4007-9faf-a1933a9a62a1" />

**Key Elements:**
* **Blue Arrow**: The original vector $x$.
* **Red Arrow**: The rotated vector $y$ (after the Givens transformation).
* **Dotted Circle**: The implicit rotation path (suggested but not drawn).

**Comments:**
* A Givens rotation zeroes out one component (for example $y[k] = 0$) while preserving the vector’s length.
* The rotation angle $\theta$ satisfies:
    * $c = \cos(\theta) = \frac{x[i]}{r}$
    * $s = \sin(\theta) = \frac{-x[k]}{r}$
    * where $r = \sqrt{x[i]^2 + x[k]^2}$
* Used in QR decomposition for selectively eliminating individual elements.

**Key Elements:**
* **Blue Arrow**: The original vector $x$.
* **Red Arrow**: The rotated vector $y$ (only the components at positions $i$ and $k$ are changed).

**Comments:**
* Rotates $x$ in the plane defined by components $i$ and $k$ (for example the $x–y$, $x–z$, or $y–z$ plane).
* All other components of the vector stay the same.
* Very efficient for sparse matrices because it only affects two coordinates at a time.
